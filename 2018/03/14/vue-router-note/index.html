细节问题

可能这时候有人想到了，如果只是把样式分开，那直接把样式文件放在<style scoped></style>内不就好了

确实这是一个解决办法，但是这里要注意对子组件的影响

  • 如果使用上述懒加载的方式解决该问题，子组件依旧可以使用该页引入的 ui 框架

  • 如果样式放在<style scoped></style>内，但又想让其影响子组件，则需要深度作用选择器
